🚀 **Excited to share my latest project: EduTrack!** 📚✨
> 
> Managing academic workflows and staying engaged can be a challenge for both students and teachers. I built **EduTrack**, a modern, feature-rich Learning Management System (LMS) designed to streamline education, enhance peer collaboration, and bring classroom management under one intuitive hub.
> 
> **Core Features:**
> * 📚 **Central Workspace:** Unified control room for video lectures, resources, assignments, and forums.
> * 🎬 **Interactive Lecture Player:** Streaming with likes and comments for collaborative studying.
> * 📝 **Peer Note Exchange:** A shared hub where students swap notes and teachers grade them (1-5 stars).
> * ❓ **Smart Q&A Corner:** Focused academic forum with image support and official solutions.
> * 🎙️ **Live Classrooms:** Live broadcasts with automated notifications and class discussions.
> * 🏆 **Student Exam Arena:** Dedicated space for students to take assessments and track progress.
> * 📊 **Teacher Dashboard:** Central panel to schedule classes, grade tasks, and track student analytics.
> * 🤖 **AI Assistant:** OpenAI-powered virtual tutor helping students with questions in real-time.
> 
> 🛠️ **Tech Stack:**
> * **Backend:** Laravel (PHP) & MySQL
> * **Frontend:** Tailwind CSS v4, JavaScript (ES6+), and Vite
> * **AI:** OpenAI API Integration
